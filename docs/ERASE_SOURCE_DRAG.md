# Erase Source Drag Hotfix

This hotfix makes the green source rectangle of an erase/clone patch draggable with the mouse.

## Behavior

- Drag the orange patch preview/target to move the area that will be covered.
- Drag the green source rectangle to choose which clean part of the base image will be cloned.
- Drag the small handle to resize the erase patch.
- Arrow keys move the selected target patch.
- Ctrl + arrow keys move the selected source rectangle.
- Shift + arrow keys still moves by 10 pixels.

The Source X and Source Y fields are still available for precise numeric adjustment.
