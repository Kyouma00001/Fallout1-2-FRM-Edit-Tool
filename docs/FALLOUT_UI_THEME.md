# Fallout-inspired UI Theme

This sprint is a visual polish pass only.

## Design goals

- evoke the Fallout 1/2 interface mood
- keep the editor practical and readable
- make the workflow easier to scan quickly
- preserve the stable export and safety workflow from the previous sprint

## Theme direction

- dark brown / black metallic panels
- amber / brass accent borders
- warm beige text
- green status for ready assets
- grouped panels similar to tool stations / terminals

## Notes

- This is an inspired theme, not an exact recreation of the original game UI.
- No asset pipeline behavior should change.
- If desired later, this can be followed by a sprint focused on iconography and finer spacing polish.
